プログラマー内で注意すべき共通事項

①命名規則---------------------------------------------------------------------------------------------
	変数名について
		例  int moveCount	int catchTestManager
		変数の名前は頭文字は小文字、以下は単語の区切りで頭文字だけ大文字。

	関数（メソッド）名
		例	void Attack()	void MovePosThis()
		各単語の頭文字を大文字。

	クラス名
		基本的に関数と一緒
	
	共通すること
		名前をつけるときは英単語で統一
		✕  int kougeki		◯  int power

②コメント---------------------------------------------------------------------------------------------
ソースを見やすくするためコメントでの説明は必ず記述する。
短く、端的、かつ分かりやすく記述する。
コメントの例は別ファイル参照。

③その他注意事項---------------------------------------------------------------------------------------------
	・public static は極力使わない。使う場合は全員に確認。
	・例外としてHP(体力)やDEF(防御)など大文字だけのが見やすいときはそちらでも可。
	　ただし頭文字は絶対に小文字で。
	　使う場合は enemyHP や myDEF hp def といった具合に名前を工夫する。
	・例外的にfor文を繰り返す用の変数などは英字１文字でも可。
	　例　for(int i = 0; i < 5; i++)  みたいな
